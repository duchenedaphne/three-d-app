# My first 3D app

This is my first real time animated 3D app

## 🛠 Stack
React JS  
Vite JS  
Tailwind CSS  
Blender

## 🛴 Get started

1 - Clone this repository :

    git clone https://github.com/duchenedaphne/three-d-app

2 - Install the dependencies

    npm install

3 - Launch the app

    npm run dev
